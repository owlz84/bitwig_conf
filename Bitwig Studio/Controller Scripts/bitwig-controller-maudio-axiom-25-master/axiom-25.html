M-Audio Axiom 25

This script was written for an Axiom 25 on factory preset 1, i.e. the state of the keyboard after a factory reset.

Rotary encoders.

The rotary encoders are by default mapped to the device macros of the primary device of the current track. You can toggle between device macros and common parameters using the drum pad C14.

Transport buttons

  • Loop: Toggles the VST window if your primary device is a VST.
  • Rewind: Selects the previous track.
  • Forward: Selects the next track.
  • Stop: Stops playback and sets the play position to the start of the arrangement.
  • Play: Starts playback of the arrangement.
  • Record: Starts recording for armed tracks.

Drum pads

  • C14: Toggles the assignment of the rotary encoders between device macros and common parameters.
